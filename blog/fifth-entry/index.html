JavaScript time:

  1. Variables. var name = "String"; or var name = 123;
  2. Print to console. console.log(name[5]);
  3. Specific behavor:
    • "Hello" + 5 * 10 will return Hello50
    • \" will return ". \n will return a new line. \t will return tab.
    • Truthy and Falsy. ' ' == false, as well as null undefined 0 NaN Returns: true. Both the operands on either side of the == operator are first converted to zero, before comparison. "Hi" == true as well as 24 true will return true.
  4. Alert messages: NaN - when the math calculation failed; undefined - when the variable has no value or when tere is no return value from a function.
  5. Strict equality. === / !==. "1" === 1 will be false, becouse "1" is a string with another code value.
  6. % - Reminder which returns the remainder left over when one operand is divided by a second operand. It always takes the sign of the dividend.
  7. Standard functions:
    • name.charCodeAt(). returns an ASCII code of a character.
    • name.charAt() or name(). Returns character of a string.
    • name.toFxed(2). Returns number with 2 decimals, e.g. 250.00.
    • Number(varName); converts string to a number, works well with toFixed, but as a string.
    • name.toLocaleString("en-US"). Passes a value with comas as per the locale "en-US.
    • name.length() - returns the length of a string or an array.
    • name.toUpperCase() - returns the value in upper case.
  8. Ternary operator. conditional ? (if condition is true) : (if condition is false). I also can include as many as I want others conditions in true or false place. Example x % y === 0 ? 1 : 2;. The ternary operator evaluates to an expression and expressions can't contain a return statement. However, you could very well return the result of a ternary operator, i.e. return name.length >= 7 ? true : false;
  9. Switch Statement. If each condition is based on the same value. switch (var name) {case 1: "action"; case 2: "action for 2nd case";}, etc. After it founds a case by bypising the variable value then it cintinues executing other cases below. To prevent that use break operator. It works with all data types. You can add a default: case to a switch statement and it will be executed when none of the values match the value of the switch expression.
  10. Operators. x *= 2 same as x = x* 2. Instead of * can me any symbol.
  11. Block scope. It is a variable that is writhen in the {} and should be declared using let instead of var, because variables declared with the `var` keyword can not have Block Scope.
  12. Variable declarations:
    • let - it is a new way to declare a variable in any scope - Global, Local, or Block. The value of this variable can be changed or reassigned anytime within its scope.
    • const - it is also a way to declare constants in any scope - Global, Local, or Block. Once you are assigned a value to a const variable, the value of this variable CANNOT be changed or reassigned throughout the code.
    • var - this is the old way of declaring variables in only two scope - Global, or Local. Variables declared with the var keyword can not have Block Scope. The value of this variable can be changed or reassigned anytime within its scope.

  17. Creating an Array. var name = ["First", "Second", "Third"]; You can even store an array in an array to create a nested array!
  18. Array Properties and Methods:
    • Length. name.length - length of the array.
    • Push. name.push("value") - push an additional value to the end of an array.
    • Pop. name.pop(); - removes the last item in the list. Also, pop() returns the element that has been removed in case you need to use it.
    • Shift. name.shift(); removes first item in the list.
    • Splice. arrayName.splice(arg1, arg2, item1, ....., itemX); where,
      • arg1 = Mandatory argument. Specifies the starting index position to add/remove items. You can use a negative value to specify the position from the end of the array e.g., -1 specifies the last element.
      • arg2 = Optional argument. Specifies the count of elements to be removed. If set to 0, no items will be removed.
      • item1, ....., itemX are the items to be added at index position arg1.
      splice() method returns the item(s) that were removed.
    • Reverse. name.reverse(); Reverses the array.
    • The forEach() loop. name.forEach(callbackFn); Executes a provided function once for each array element. The function is called with the following arguments:
      • element

        The current element being processed in the array.

      • index

        The index of the current element being processed in the array.

      • array

        The array forEach() was called upon.

      The forEach() method gives you an alternative way to iterate over an array, and manipulate each element in the array with an inline function expression.

      Notice that the forEach() method iterates over the array without the need of an explicitly defined index. In the example above, donut corresponds to the element in the array itself. This is different from a for or while loop where an index is used to access each element in the array.

      E.g.: 

      words = ["cat", "in", "hat"];
words.forEach(function(word, num, all) {
    console.log("Word " + num + " in " + all.toString() + " is " + word);
});
                                
      Prints:
Word 0 in cat,in,hat is cat
Word 1 in cat,in,hat is in
Word 2 in cat,in,hat is hat

    • The map() loop. Works the same as forEach() loop, but creates and returns a new array populated with the results of calling. So it is a good idea to store it in a variable, using function expression. Just include a return keyword with a value inside a function.
    • ... Others like sort, join, concat, etc., see threw the link: Methods for modifying arrays.
